# gorilla-practice

Tech stack
- Javascript
- Cypress


# Installation
Using your terminal, navigate to your project's folder, then run the following commands
git clone https://github.com/josequesadaGL/gorilla-practice.git
npm install

# Running
npm run cypress:test
